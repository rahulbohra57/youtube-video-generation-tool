# This is v2 update of the Autoframe App

The app works fine. Now I want to integrate below updated functionality to the system and automate things.

Use /superpowers:brainstorm and give me a plan to integrate following functionality:

## 1. Lead Researcher: 
A smart agent which will scan "Google News" after every set timer (let's take 4 hours for starting). The scanning will be done to fetch ongoing events in user selected domains like Tech, Science, Politics, Sports, Current Affairs, etc. User can select 4-5 genres and the agent will only fetch brief information with a headline about top news in those genres. The Lead Researcher is responsible for shortlisting top 4-5 news per genre basis the virality of the news and give a Rating of itself out of 5 to every news. Then . Lead Researcher will then share the top 5 outputs basis the ratings with Whatsapp Agent.

## 2. WhatsApp Agent: [Agents Required: Lead Researcher]
This  agent handles all the communication with user. It will send a whatsapp message to the user. The message will contain all the brief information about the news shared by Lead Researcher. It should strictly follow the below format to share the news in whatsapp message:
{
**Unique Number:** [A number assigned to the news by Whastapp Agent which user will refer to for replying. It should be in this format: [Genre][News No.]. For eg. TECH01 ]
**Genre:** [Which genre the news belongs to]
**Headline:** [A strong headline]
**Context:** [A small brief about the news]
**Rating:** [Rating given by Lead Researcher]
}

Once user replies with the Unique Number like TECH01, the WhatsApp Agent will pass Headline to Generator Agent. If the user replies with None, the pipeline should end until next set of news is sent by the Lead Researcher after the set timer (typically 4-5 hours).

## 3. Generator Agent: [Agents Required: WhatsApp Agent]
This agent will take the headline from the Whatsapp Agent and put in the app. It is responsible to generate the content using the AUTOFRAME App. It should always generate content in "English" and "9:16 ratio" for YouTube Shorts. It will also generate the post caption using the app. Once the output is generated from the app. It will send the output video and YT shorts post caption to the Social Media Agent.

## 4. Social Media Agent: [Agents Required: Generator Agent, WhatsApp Agent]
This agent is responsible for ensuring the post caption is suitable for YouTube Shorts and if required, will do changes like adding a hook, add engaging ending asking users to like and subscribe in the post caption. This agent is also responsible for posting content on YouTube page through OAuth 2.0 access. It should mandatorily add suitable title and post caption in the YouTube post. Once posted, it will reply posted to the Whastapp Agent in below format:

{
**Post Title:** [Title of the video]
**Post Link:** [Live link to view the post]
}
